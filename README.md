# Projeto P1 - CC3642
Projeto de Orientação a Objetos

Documentação em: dist/javadoc/index.html

Para dar a sensação de movimentação, deixe a janela de saída do netbeans no canto de baixo da tela, e a puxe até o topo da tela.

O tempo de sleep é uma variável, e a primeira coisa que o programa pede ao iniciar é que o usuário digite o tempo de sleep em ms. O programa rodou fluido com um Thread.sleep(1000) em todas versões.
